Dashboard KPI’s overflow
★ Objectifs
➔ Analyser les données de flux téléphonique et la ventilation des appels vers les hôtels : données Colt
➔ Puis après le débordement vers la réservation centrale : données Webhelp
➔ Ainsi que les réservations générées grâce au débordement d’appel vers la centrale de réservation : données Réservation
Liste des abréviations
  • SVI (Serveur Vocal Interactif) Système automatisé qui guide les appelants via un menu vocal
  • VDN (Vector Directory Number) Numéro virtuel utilisé pour diriger les appels entrants selon des règles définies
  • ACW (After Call Work) Temps que prend un agent pour finaliser le traitement d’un appel après avoir raccroché
  • ACD (Automatic Call Distributor) Système qui distribue automatiquement les appels entrants vers les agents disponibles les plus appropriés
  • AHT (Average Handling Time) Durée moyenne nécessaire pour traiter un appel
  • AVG (Average) Moyenne
